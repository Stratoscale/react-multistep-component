## React Wizard Steps

`react-wizard` is a step component for React :love:

### Installation

Using npm
```
npm install react-wizard
```

### Demo

```js

import { Steps, Step } from 'react-wizard';
{
  /*
   * <Steps> bla bla
   *
   */
}
<Steps currentStep=1>
  {
    /*
     * <Step> bla bla
     *
     */
  }
  <Step>
    BlaBla
    <span>Hola</span>
  <Step/>
  <Step>
    BlaBla
    <span>Hola</span>
  <Step/>
</Steps>

```
