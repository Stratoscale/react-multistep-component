## React Multistep Component

`react-multistep-component` is a step component for React :love:

### Installation

Using npm
```
npm install react-multistep-component
```

### Demo

```js

import { Steps, Step } from 'react-multistep-component';
{
  /*
   * <Steps> bla bla
   *
   */
}
<Steps currentStep={1}>
  {
    /*
     * <Step> bla bla
     *
     */
  }
  <Step>
    BlaBla
    <span>Hola</span>
  <Step/>
  <Step>
    BlaBla
    <span>Hola</span>
  <Step/>
</Steps>

```
